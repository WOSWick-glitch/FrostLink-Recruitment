# FrostLink

A premier recruitment platform for Whiteout Survival. Find the perfect state, join elite alliances, or recruit top-tier talent for your roster.

## Features
- **Player Profiles:** Detailed character sheets including power buckets, furnace levels, and endorsements.
- **Alliance Directory:** Browse top alliances currently recruiting, complete with requirements and open slots.
- **State Directory:** Explore states, their rules, and top alliances to find the best fit for your playstyle.

## Development

This is a frontend prototype built with:
- React
- Vite
- Tailwind CSS
- Wouter for routing
- Shadcn UI components

### Getting Started

To run the development server:

```bash
npm install
npm run dev
```

The application will be available on port 5000.
